猫嘻VR游戏平台品牌VI设计
客户：猫嘻(北京)科技有限公司 / 项目:LOGO设计/VI设计
VR即Virtual Reality，意为虚拟现实技术，虚拟现实技术，利用电脑虚拟构建场景，通过佩戴设备即可体验置身虚拟环境。VR领域在目前市场大致可以分为VR硬件、VR软件，以及平台型VR服务商。

喵嘻（北京）科技有限公司是国内领先的VR游戏运营商，致力于VR尖端技术的研发以及VR游戏的普及和推广。喵嘻（miaoxi.com）旨在打造国内最优秀的VR游戏平台，因此品牌的构建趋向于技术与商业的有机融合，并最终将喵嘻打造成一个“最好玩的VR游戏互动平台”。

喵嘻的主体标志图形概括了猫的形象，并将VR眼镜以及传统的游戏手柄等视觉元素融入其中，并最终形成了一个具有强烈未来感的视觉符号。
